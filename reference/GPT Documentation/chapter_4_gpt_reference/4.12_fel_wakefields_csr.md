The FEL module allows multi-mode, multi-pass FEL simulations, with emphasis on phase-space evolution of the electron bunch during passage through the undulator. This is an expert element for expert users. If you plan to use GPT for serious design work on FELs, please contact us for additional advice.

The wakefield element calculates the fields of longitudinal and transverse wakefields in a 1D approximation. The wakefunctions are read from an external tabulated file and this file needs to be precomputed using other software. The implementation for the wakefields is not based on a series of kicks, but the wakefields themselves are added to the equations of motion. This is more natural and more accurate in a time-domain code, but please note that the units of the wakes are different compared to most other codes.

The CSR module is a full-featured CSR model based on 1D segmentation of the bunch in 3D space. The fields are directly evaluated from the Li├®nardŌĆōWiechert potentials, based on the stored history of the bunch. This allows for example for microbunching studies in a series of magnets close together where the entrance and exit effects are crucial. There is an embargo on this element for a specific use case. Please contact us for details.
