- **`L`** вАФ Length of the element [m].

- **`Gmin`** вАФ Minimum Lorentz factor.

- **`Gmax`** вАФ Maximum Lorentz factor.

- **`L`** вАФ пАЉ *z* . In that range, it removes all particles with a Lorentz factor outside the range bracketed by the specified `Gmin` and `Gmax`.

### 4.10.2 Multislit

```
multislit(ECS,a,b,L,d,N) ;
```
Removes a particle if it does not pass through an opening of a multiple slit.

- **`ECS`** вАФ Element Coordinate System.

- **`a`** вАФ Width of a slit opening [m] (in **x**-direction).

- **`b`** вАФ Height of a slit opening [m] (in **y**-direction).

- **`L`** вАФ Length of the element [m] (in **z**-direction).

- **`d`** вАФ Distance between the centers of two slit openings [m].

- **`N`** вАФ Number of slits.

This element simulates a multiple slit. The particles can only pass through the openings of the slit, all other particles are removed if

- **`L`** вАФ пАЉ *z* . This element is local.

### 4.10.6 Zminmax

```
zminmax(ECS,zmin,zmax) ;
```
Removes all particles in front of zmin or behind zmax.

- **`ECS`** вАФ Element Coordinate System.

- **`zmin`** вАФ Minimum *z*-coordinate [m].

- **`zmax`** вАФ Maximum *z*-coordinate [m].

The particle is removed when *z*<`zmin` or *z*>`zmax`.
