## 2.7 Magnetic mirror

As a demonstration of the high accuracy achievable with GPT a magnetic mirror is simulated. A magnetic field gradient reflects a charged particle gyrating around the magnetic field lines. The field gradient is generated by a solenoid located at *z*=0 with a radius of 20 cm and 100 loops of 100 A.

**Listing 2-14:** GPT inputfile for a magnetic mirror: `tutorial/magmir/magmir.in`.

```

# Ultra-high accuracy for best results

# Define beam parameters

# Start the particle
setstartpar( "beam", 0,Rl,-1, GBetax,0,GBetaz ) ;

# Position the solenoid
solenoid( "wcs", "I", 0.2,10000 ) ;

# Time output for trajectories and theta-scan
snapshot( 0,0.7e-6,0.5e-9) ;
# snapshot( 1e-6 ) ;

```
