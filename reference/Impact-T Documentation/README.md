# IMPACT-T User Document Version 3.1

Ji Qiang — Lawrence Berkeley National Laboratory

Copyright © The Regents of the University of California. LBNL-62326, 2007.

IMPACT-T is a fully three-dimensional program to track relativistic particles taking into account space charge forces, short-range longitudinal and transverse wakefields, and coherent synchrotron radiation (CSR) wakefield. It is one of the few codes used in the photoinjector community with a parallel implementation, making it very useful for high statistics simulations of beam halos and beam diagnostics.

This markdown-based documentation was generated from [ImpactTv3.pdf](ImpactTv3.pdf) and may have issues with transcription of equations or formatting. If there are errors, please refer to the original PDF.
