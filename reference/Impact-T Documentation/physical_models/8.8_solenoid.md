# 8.8 Solenoid

In the IMPACT-T code, by using type code "3", the magnetic fields of solenoid `Br` and `Bz` are read in from an external file containing discrete data on r-z plane. By using type code "105", the magnetic fields of the solenoid can be obtained from the on-axis field data described by Fourier coefficients as:

```
Bx = -B'0*x/2 + B'''0*x*(x² + y²)/16

By = -B'0*y/2 + B'''0*y*(x² + y²)/16

Bz = B0 - B''0*(x² + y²)/4
```

where `B'0`, `B''0`, `B'''0` are the first, the second and the third derivatives of field on axis.
