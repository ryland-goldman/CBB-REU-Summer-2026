# 8.5 Longitudinal CSR Wakefield

The longitudinal CSR wake can be included in the simulation together with the 3D space-charge calculation if the first line input parameter in the input `rfdata` file is greater than 0.

> *Note: the following CSR model is no-longer used in the IMPACT-T code. The current version uses an integrated Green function CSR model described in the references [5, 6].*

Under the ultra-relativistic approximation, the rate of change of energy in the bunch is given by [7, 8]:

```
dE(s,φ)/(c*dt) = -(2e²) / (4πε0 * 3^(1/3) * R^(2/3)) * 
                  (∫[s-sL to s] (1/(s-s')^(1/3)) * ∂λ(s')/∂s' ds'
                   + (λ(s-sL) - λ(s-4sL)) / sL^(1/3))
```

where `R` is the bend radius, `φ` is the angle into the bend, `s` is the relative position within the bunch, `t` is time, `sL = Rφ³/24` is the slippage length. The first term is normally called the steady-state wake when the slippage length becomes longer than the bunch length. The second term denotes the entrance transient wake.

For a bunch moving out of the bend, the CSR wake function under ultra-relativistic approximation are given by [9]:

```
W(s) = -4/R * (1/(φm+2x)) * λ(s - R/6*φm²*(φm+3x))          (for source in front of the bend)

W(s) = 4/R * ((λ(s-Δsmax)/(φm+2x)) + ∫[s-Δsmax to s] (1/(φm+2x)) * ∂λ/∂s' ds')   (for source inside the bend)
```

where dimensionless `x (x/R -> x)` is the distance from the observation point to the exit of the bend magnet, `φm` is the total bend angle of the magnet, `ψ` is related to `s - s'` through the equation:

```
s - s' = (R*ψ³/24) * (ψ + 4x) / (ψ + x)
```

and the `Δs_max` is defined by:

```
s - s' = (R*φm³/24) * (φm + 4x) / (φm + x)
```

In the IMPACT-T code, the line density function `λ` and its derivatives are smoothed using a first-order Savitzky-Golay filter [10]. The first derivatives of the density function is further smoothed using a customer designed local filter to calculate the second derivative [11].
