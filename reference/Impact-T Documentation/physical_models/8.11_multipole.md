# 8.11 Multipole

In the IMPACT-T code, the multipole element includes sextupole, octupole and decapole.

## Sextupole

```
Bx = B0 * x * y
By = B0 * (x² - y²) / 2
```

## Octupole

```
Bx = B1 * (3x²y - y³) / 6
By = B1 * (x³ - 3xy²) / 6
```

## Decapole

```
Bx = B2 * (x³y - xy³) / 6
By = B2 * (x⁴ - 6x²y² + y⁴) / 24
```

where `B0`, `B1`, and `B2` are input parameters.
