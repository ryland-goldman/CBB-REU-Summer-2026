6.3 beamlossntuple NTuple containing particle tracks when lost.

All possible loss mechanisms are included. Tune particle tracks
are omitted, but Reference track(s) will appear if one or more
Reference particles are tracked.

This command is not placed into the geometry.

The NTuple contains the usual track data:

The standard NTuple fields are:
x,y,z (mm)
Px,Py,Pz (MeV/c)
t (ns)
