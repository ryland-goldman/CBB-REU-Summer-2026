## 7.12 triplet.sh – tune a quad triplet for point-to-point focus

This shell script requires the bash shell and the following programs:
• Gbeamline (http://g4beamline.muonsinc.com )
• gminuit (http://muonsinc.com ComputerPrograms/gminuit)
• gnuplot (available from your Linux distribution or http://sourceforge.net )
• tclsh and wish (available from your Linux distribution)

The gminuit program is configured to have the following parameters: P, dP, sigmaXp, sigmaYp, QF,
QD. The first four determine the initial beam (which emanates from a point), and the last two determine
the gradients of the three quads. The Chisquared to be minimized is (sigmaX^2+sigmaY^2) evaluated at
Z=6000, so the minimum Chisquared will be a point focus. Because of the front-to-back symmetry, the
two horizontal-focusing quads have the same gradient.

buttons to execute the script once and to fit the parameters (see the gminuit documentation for details):
