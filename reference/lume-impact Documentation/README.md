# LUME-Impact

Python interface for running IMPACT-T and IMPACT-Z within the LUME framework. Provides a unified API for input configuration, simulation execution, output parsing, and visualization. Source repository: [lume-impact on GitHub](https://github.com/ChristopherMayes/lume-impact).
