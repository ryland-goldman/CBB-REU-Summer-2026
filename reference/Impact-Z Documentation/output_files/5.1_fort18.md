## 5.1. File fort.18: Reference Particle Information

This file contains information about the reference particle as a function of distance along the beamline.

| Column | Description |
|--------|-------------|
| 1st | distance (m) |
| 2nd | absolute phase (radian) |
| 3rd | gamma |
| 4th | kinetic energy (MeV) |
| 5th | beta |
| 6th | Rmax (m) — R is measured from the axis of the pipe |
