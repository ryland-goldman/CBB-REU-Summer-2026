## 5.2. File fort.24 (for X), fort.25 (for Y), fort.26 (for Z): RMS Size Information

These files contain RMS beam size information as a function of position along the beamline.

- **fort.24** — transverse X plane
- **fort.25** — transverse Y plane
- **fort.26** — longitudinal Z plane

| Column | Description |
|--------|-------------|
| 1st | z distance (m) |
| 2nd | centroid location (m) |
| 3rd | RMS size (m) — degree for fort.26 |
| 4th | centroid momentum (rad for fort.24 and fort.25, MeV for fort.26) |
| 5th | RMS momentum (rad for fort.24 and fort.25, MeV for fort.26) |
| 6th | Twiss parameter alpha |
| 7th | normalized RMS emittance (m-rad for transverse; degree-MeV for fort.26) |
