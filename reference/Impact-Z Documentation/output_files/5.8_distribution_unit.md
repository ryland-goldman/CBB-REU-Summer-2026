## 5.8. Distribution Unit

When using the `"-2"` type code (described in the [lattice section](../input_files/1.1.2_lattice_section.md)) to print particle data at various locations, the resulting data will be in columns of the form:

```
x, px, y, py, t, pt, q/m, charge/per particle, id
```

The data are dimensionless in IMPACT internal units:

---

| Column | Description |
|--------|-------------|
| **1st** | x normalized by c/omega |
| **2nd** | Px normalized by mc — must be divided by gamma\*beta to convert to unit radian |
| **3rd** | y normalized by c/omega |
| **4th** | Py normalized by mc — must be divided by gamma\*beta to convert to unit radian |
| **5th** | phase (radian) |
| **6th** | kinetic energy deviation (E0 - E) normalized by mc² |
| **7th** | q/m — for an electron, q = -1, m = 0.511001e6 |
| **8th** | charge per macro-particle (Coulomb per particle) |
| **9th** | particle id |
