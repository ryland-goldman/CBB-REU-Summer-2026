## 5.3. File fort.27: Maximum Amplitude Information

This file contains maximum amplitude information of the beam distribution as a function of position.

| Column | Description |
|--------|-------------|
| 1st | z distance (m) |
| 2nd | Max. X (m) |
| 3rd | Max. Px (rad) |
| 4th | Max. Y (m) |
| 5th | Max. Py (rad) |
| 6th | Max. Phase (degree) |
| 7th | Max. Energy deviation (MeV) |
