## 5.4. File fort.28: Load Balance and Loss Diagnostic

This file contains load balance and particle loss diagnostics as a function of position.

| Column | Description |
|--------|-------------|
| 1st | z distance (m) |
| 2nd | min # of particles on a PE |
| 3rd | max # of particles on a PE |
| 4th | total # of particles in the bunch |
