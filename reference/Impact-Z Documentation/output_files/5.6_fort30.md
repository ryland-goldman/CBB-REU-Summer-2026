## 5.6. File fort.30: Square Root of 4th Moments of the Beam Distribution

This file contains the square root of the 4th moments (kurtosis measure) of the beam distribution as a function of position.

| Column | Description |
|--------|-------------|
| 1st | z distance (m) |
| 2nd | X (m) |
| 3rd | Px (rad) |
| 4th | Y (m) |
| 5th | Py (rad) |
| 6th | phase (degree) |
| 7th | Energy deviation (MeV) |
