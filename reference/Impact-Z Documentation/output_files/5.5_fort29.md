## 5.5. File fort.29: Cubic Root of 3rd Moments of the Beam Distribution

This file contains the cubic root of the 3rd moments (skewness measure) of the beam distribution as a function of position.

| Column | Description |
|--------|-------------|
| 1st | z distance (m) |
| 2nd | X (m) |
| 3rd | Px (rad) |
| 4th | Y (m) |
| 5th | Py (rad) |
| 6th | phase (degree) |
| 7th | Energy deviation (MeV) |
