## 2.1. Run IMPACT-Z

To run IMPACT-Z using MPI with 16 processors:

```
mpirun –n 16 ImpactZexe
```

The number of processors must equal the product of the processor layout dimensions specified on line 1 of `ImpactZ.in`. For example, if line 1 is `4 4`, you must use `4 × 4 = 16` processors.
