## 2.3. Stop IMPACT-Z

To stop a running IMPACT-Z simulation, press:

```
Ctrl + C
```

The simulation will terminate at the current step. Output files written up to that point will be retained.

> **Tip:** To run only partway through a large input file without stopping mid-simulation, use the [`-99` element type](../input_files/1.1.2_lattice_section.md) in the lattice section to halt execution at a specific point.
