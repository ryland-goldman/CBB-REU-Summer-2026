## 1.1.1. Beam Section

The beam section consists of 11 lines. Using the example `test.in`:

```
① 4 4
② 6  100000  2  0  1
③ 64 64 64  1  0.1  0.1  0.1
④ 19  0  0  1
⑤ 100000
⑥ 0.04
⑦ 1.0657886728e-09
⑧ 0.011389835258    0.000121257352747    0.897613531029    1.0    1.0    0.0    0.0
⑨ 0.0239487992104   0.000130507846319   -0.0704357946046   1.0    1.0    0.0    0.0
⑩ 0.0501516670178   0.0000106577886728   0.287347885566    1.0    1.0    0.0    0.0
⑪ 0.04   172359000.0   938272310.0   1.0   650000000.0   110.163
```

---

### Line-by-Line Description

**Line 1:** Processor layout in y and z dimensions. The product of these numbers must equal the number of processors (4, 4) that you run on.

**Line 2:** 
- `"6"` — random seed of simulation
- `"100000"` — macro particles used in simulation
- `"2"` — nonlinear Lorentz integrator (`"1"` for linear map integrator)
- `"0"` — no error study (`"1"` for error studies)
- `"1"` — standard output (`"2"` for 90%, 95%, 99% emittance, radius output; otherwise no output)

**Line 3:** `64x64x64` space-charge grid, followed by boundary condition flag and pipe dimensions.
- `"1"` — 3D open (all mesh numbers must be a power of 2, i.e. 2^n)
- `"2"` — transverse open, longitudinal periodic (3rd mesh number must be 2^n+1)
- `"3"` — transverse finite, longitudinal open round pipe (2nd mesh number must be 2^n+1)
- `"4"` — transverse finite, longitudinal periodic round pipe (2nd and 3rd mesh numbers must be 2^n+1)
- `"5"` — transverse finite, longitudinal open rectangular pipe (1st and 2nd mesh numbers must be 2^n+1)
- `"6"` — transverse finite, longitudinal periodic rectangular pipe (all mesh numbers must be 2^n+1)

> **Note:** In the current version, only 3D open BC is available. x pipe width = 0.1 m, y pipe width = 0.1 m, period length = 0.10 m.

**Line 4:** Distribution type and options.
- `"19"` — distribution type. Distribution types:
  - `1` = rectangle in 6D phase space (single charge)
  - `2` = 6D Gaussian (single charge)
  - `3` = 6D Waterbag (single charge)
  - `4` = Semi-Gaussian (uniform position, Gaussian momentum, sc)
  - `5` = KV transverse, uniform longitudinal (sc)
  - `16` = 6D Waterbag for multiple charge state (mc)
  - `17` = 6D Gaussian for multiple charge state (mc)
  - `19` = Read coords/momenta from external file `"particle.in"` in ImpactZ format
  - `22` = Read coords/momenta from external file `"particle.in"` in Elegant format
  - `35` = Read coords/momenta from external file `"particle.in"` in ImpactT output format
  - `199` = Parallel read in macroparticles from `"fort.x+myid"` files (x is an input positive integer from the 2nd number of the initial distribution parameter line, myid is the processor ID; all files are in binary format with the # of particles nptlocal as the 1st line, the particle data pts(1:9,1:nptlocal) as the second line)
- `"0"` — no restart (`"1"` = restart from some point after stop)
- `"0"` — no sub-cycle (`"1"` = sub-cycle)
- `"1"` — number of charge states = 1

**Line 5:** Particle list for each charge state. Example: `1000000`.

**Line 6:** Current for each charge state. Example: `0.04` A.

**Line 7:** q/m for each charge state. Each charge is normalized by the mass of the reference particle so that the reference particle has 1 AMU mass but less charge. For example, for a proton: `1 / 938.23e6 = 1.0657886728e-09`.

**Lines 8–10:** Twiss parameters of the initial distribution. The three lines correspond to the three planes:

```
Alpha_x    beta_x    emit_x    mismatchx    mismatchpx    offsetX    offsetPx
Alpha_y    beta_y    emit_y    mismatchy    mismatchpy    offsetY    offsetPy
Alpha_z    beta_z    emit_z    mismatchz    mismatchE     offsetPhase  offsetEnergy
```

The distribution in each plane is a quadratic form. There is no coupling among planes. `Alpha_` and `beta_` (in meters for x and y, and degree/MeV for z) are Twiss parameters. `emit_` is normalized emittance (in m-rad for x and y, and degree-MeV for z).

**Line 11:** 
- Current averaged over RF period (e.g. 0.04A, I=Q\*freq)
- Initial kinetic energy (e.g. 172.359 MeV)
- Particle mass (e.g. 938.272 MeV/c²)
- Charge (e.g. 1)
- Scaling frequency (e.g. 650 MHz)
- Initial phase of the reference particle (e.g. 110.163)
