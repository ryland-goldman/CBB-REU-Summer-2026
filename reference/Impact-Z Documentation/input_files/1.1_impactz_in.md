## 1.1. ImpactZ.in

The main IMPACT-Z input file is called `ImpactZ.in`. The twelfth line of the file starts with `!` which means it is a comment line. This line divides the file into two sections: the [beam section](1.1.1_beam_section.md) and the [lattice section](1.1.2_lattice_section.md). The code will not read this line.

> **WARNING:** *The `/` at the end of each line in the lattice section is important.* The read statement in IMPACT-Z reads all the beamline element info and wants to read a lot of numbers. In a Fortran free-format read statement, you can make it ignore the rest by using `/` at the end of the line.
>
> If you neglect this, the code will crash. Also, any line starting with `!` will be treated as a comment line.
